# EMH - Escola de Música Harmonia

## 📋 About the Project

**EMH — Escola de Música Harmonia** is a web-based platform for music management and teaching 
developed for the Escola de Música Harmonia, an academic institution for music education 
located in Maia, Portugal, under the direction of Prof. Anabela Ferreira.

The project arose from the need to provide the institution with a solid digital presence and 
a centralised, modern and accessible learning environment. The platform incorporates 
Learning Management System (LMS) features, enabling student management, 
the provision of educational content, including sheet music, audio and 
video recordings, the monitoring of each student’s progress and communication between 
teachers and students.

## 🛠️ Technologies Used

* **Python** v3.14.5
* **Django** v6.0.6
* **PostgreSQL** v18.x
